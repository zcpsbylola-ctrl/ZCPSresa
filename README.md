# Ze Agenda Lola

## Installation
1. Créer un Google Sheet avec un onglet `bookings`.
2. Créer un Google Calendar pour les réservations.
3. Copier `apps-script/code.gs` dans un projet Apps Script et publier en tant qu'API Web App.
4. Remplacer `SHEET_ID` et `CALENDAR_ID` dans `code.gs`.
5. Remplacer `YOUR_SCRIPT_ID` dans `script.js`.
6. Pousser le projet sur GitHub. Accessible en PWA sur mobile.

## Fonctionnalités
- Service 25mn + buffer 70mn.
- Blocage automatique des créneaux réservés.
- Mobile-first, scroll horizontal.
- Validation WhatsApp après réservation.
